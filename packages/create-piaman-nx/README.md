# create-piaman-nx

> The official scaffolding initializer for setting up new Piaman-NX projects in seconds.

`create-piaman-nx` is a lightweight project initializer designed to set up a fully configured, production-ready Piaman-NX backend application structure without any manual configuration.

## Usage

You don't need to install this package globally. Simply run it using your favorite package manager's initializer command:

### Using `npm`

```bash
npm create @piaman-nx@latest my-awesome-app
```
