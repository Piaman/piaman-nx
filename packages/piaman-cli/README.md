# @piaman/piaman-cli

> The official Command Line Interface (CLI) for the Piaman-NX backend framework.

`@piaman/piaman-cli` provides a powerful set of terminal tools designed to speed up your development workflow. It handles project initialization, scaffolding components, and running your local development environment.

## Installation

You can install the CLI globally to access the `piaman` command from anywhere:

```bash
npm install -g @piaman/piaman-cli
```
